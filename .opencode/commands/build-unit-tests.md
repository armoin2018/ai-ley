# build-unit-tests

Generate comprehensive unit testing infrastructure including test configuration, health checks, regression tests, synthetic transactions, coverage dashboards, and CI/CD integration.

## Usage

```bash
ai-ley build-unit-tests [--mode=plan|dry-run|apply]
```

## Full Documentation

See: `.ai-ley/shared/prompts/build-unit-tests.md`

This command creates:

- Unit test structure and configuration
- Health check tests for system components
- Regression tests with golden files
- Synthetic transaction journeys
- Test Control API for remote execution
- Coverage dashboards and policies
- Timeout/retry mechanisms
- CI/CD workflow integration
